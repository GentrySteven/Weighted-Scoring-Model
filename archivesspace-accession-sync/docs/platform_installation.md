# Platform Installation Guide

## Windows
1. Download from [python.org](https://python.org/downloads/)
2. **Check "Add Python to PATH"** during installation
3. Open Command Prompt: `python --version`

## macOS
With Homebrew: `brew install python@3.12`
Or download from [python.org](https://python.org/downloads/)

## Linux
Ubuntu/Debian: `sudo apt install python3.12 python3.12-venv python3-pip`
Fedora: `sudo dnf install python3.12`
